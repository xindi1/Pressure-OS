# Pressure OS v2

A mobile-first PWA for tracking pressure load across four domains.

## Four domains

1. Financial
2. Health
3. Relationships
4. Responsibilities

## Migration

Legacy Pressure OS v1 domains are automatically mapped:
- Interpersonal → Relationships
- Professional → Responsibilities
- Situational → Responsibilities
- Financial → Financial
- Health → Health

## Features

- Status history
- Import/export/clear
- Dark mode
- 7-day pressure changes
- Local storage
- New compression/load icon set
