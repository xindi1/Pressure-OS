# Pressure OS v1 Full

A Foundation-style mobile-first PWA for tracking pressures that consume mental bandwidth.

## Core Question
What is currently exerting pressure on my life?

## Types
- Financial
- Health
- Interpersonal
- Professional
- Situational

## Owners
- Me
- Partner
- Child
- Parent
- Pet
- Shared
- Other

## Statuses
- Active
- Monitoring
- Reducing
- Resolved

## Features
- Status history
- Import/export/clear
- Dark mode
- 7-day pressure changes
- Local storage
